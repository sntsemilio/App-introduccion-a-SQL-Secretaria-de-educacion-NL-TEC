# App-introduccion-a-SQL-Secretaria-de-educacion-NL-TEC
Aprende SQL Fácil es una aplicación web interactiva desarrollada en Streamlit que permite enseñar los fundamentos de SQL a personas sin experiencia previa en programación o bases de datos. La aplicación combina teoría, ejemplos prácticos y ejercicios interactivos para crear una experiencia de aprendizaje completa y efectiva.
