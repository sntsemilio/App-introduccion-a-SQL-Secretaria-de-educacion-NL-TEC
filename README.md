# Aprende SQL - Tutorial Interactivo

Una aplicación web construida con Streamlit para enseñar SQL a principiantes sin experiencia previa en programación o ciencia de datos.

## Características

- **Tutorial paso a paso**: 7 lecciones progresivas que cubren los conceptos fundamentales de SQL
- **Ejemplos interactivos**: Cada concepto incluye ejemplos que puedes ejecutar inmediatamente
- **Ejercicios prácticos**: Actividades para reforzar el aprendizaje
- **Base de datos de ejemplo**: Datos realistas de empleados y departamentos
- **Zona de práctica libre**: Espacio para experimentar con consultas propias
- **Interfaz amigable**: Diseñada específicamente para principiantes

## Contenido del curso

1. **Introducción a SQL**: Conceptos básicos y terminología
2. **SELECT**: Consultas básicas para seleccionar datos
3. **WHERE**: Filtrar información con condiciones
4. **ORDER BY**: Ordenar resultados
5. **COUNT y GROUP BY**: Contar y agrupar datos para análisis
6. **JOIN**: Combinar información de múltiples tablas
7. **Práctica libre**: Zona de experimentación
